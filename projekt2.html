Sem1 - tema 2

Grundlæggende web


Opgave: Producer et responsivt website med et konsistent udtryk, som følger grundlæggende webkonventioner og har valid kode.

Beskrivelse

Proces: I første modul arbejdede vi gruppevis med research inden for en given stilart og præsenterede herefter vores viden for resten af holdet. Dette efterfulgt af en idegeneringsfase, hvor jeg udarbejdede et moodboard samt et mock-up ud fra en udleveret wireframe. I andet modul opsatte jeg en simpel mobiludgave af stilartsitet med html og css, så den tilpassede sig desktop efter et udleveret layoutdiagram.

I tredje modul arbejdede vi med websidens visuelle udtryk - jeg traf valg om farver, font og visuelle virkemidler mv. Jeg designede desuden et ikon og et splashbillede. I fjerde modul færdiggjorde jeg mit website, gjorde det responsivt og implementerede de valgte fonte, farver og ikoner mm.

Løsning: Dette resulterede i et website om stilarten Swiss Style. Jeg tog udgangspunkt i stilens grundprincipper om minimalisme og læselighed. Jeg valgte derudover at gå med en harmonisk og enkel farvepalette.


Arbejdet med:

Html og css / Flexbox Mediaqueries / Digital indholdsproduktion / Komposition og layout, herunder gestaltlove og kontraster / Layoutprincipper, gridbaseret layout / Responsivt webdesign / Content strategi - mobile first / Billedbehandling i Photoshop - beskæring og eksport i forskellige formater / Opsætning af tekst og billeder i Adobe XD / Grundlæggende typografi - google fonts, webfonte og fontface, tekstopsætning og font-pairing / Grundlæggende farvelære - farvesystemer og kontraster mm. / Digitale udvekslingsformater - bitmap vs. vector / Arbejdsfiler vs. filformater optimeret til web / Modellering, stukturering og udvikling af brugergrænseflader - mappe- og filstruktur, navigation, navngivning, html dokumentstruktur / Teknologier brugt i udviklingen af brugergrænsefladen: Webdomæne, mappestruktur, client-server, Filezilla, browser inspector, Brackets, Adobe Xd, Adobe Photoshop / Git og versionsstyring / Idegenerering, moodboard, skitsering, wireframes, mock-ups, styletile
